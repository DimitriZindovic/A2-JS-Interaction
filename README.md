<h1>Projet Webshop Pokémon</h1>

<h2>Description :</h2>

Il s'agit d'un webshop simple pour les Pokémon utilisant l'API Pokémon. Le site comprend plusieurs pages offrant différentes fonctionnalités aux utilisateurs

<h3>Page d'accueil :</h3>

<ul>
  <li>Consulter les détails d'un Pokémon</li>
</ul>

<h3>Page de détails :</h3>

<ul>
  <li>Ajouter un Pokémon au panier (avec une quantité)</li>
</ul>

<h3>Page du panier :</h3>

<ul>
  <li>Visualiser le panier</li>
  <li>Payer le panier (le vider)</li>
</ul>

<h2>Technologies :</h2>

Symfony
Bootstrap
